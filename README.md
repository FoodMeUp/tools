# Tools

## Installation

1. Install project `yarn install`
2. Create env file `cp .env.origin .env`
3. Fill `.env` file
4. choose your command below ;)

## Utility commands

| Command       | Description                |
| ------------- | -------------------------- |
| `yarn pretty` | Run prettier to clean code |

## Tool commands

| Command                          | Description                |
| -------------------------------- | -------------------------- |
| `yarn convert:jsonToProductBulk` | Run prettier to clean code |
